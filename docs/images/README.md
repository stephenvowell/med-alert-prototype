# Documentation images

| File | Purpose |
|------|--------|
| `medalert-setup-portal.svg` | Reference wireframe of the **captive setup** page (`http://192.168.4.1/`). |
| `medalert-lan-dashboard.svg` | Reference wireframe of the **LAN dashboard** (`http://<device-ip>/`). |

These are **vector wireframes** aligned with the HTML in `src/main.cpp`, not photographs of a phone screen. They stay readable when zoomed and track UI changes better than stale PNGs.

To use **real screenshots** instead: capture from a device, save as `*.png` in this folder, update the `![…](…)` links in [`USER-GUIDE.md`](../USER-GUIDE.md), and **avoid** leaking Wi‑Fi passwords or Twilio tokens in images (see USER-GUIDE privacy section).
